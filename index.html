FoalTS

Start fast, code less, do more.

FoalTS is a high-level Node.JS framework to quickly build web apps in TypeScript. By providing a robust architecture with a large amount of packages and tools, FoalTS is designed to let you fastly bootstrap and develop ambitious backend projects.

Get started

Create your first app

$ npm install -g yo generator-foal
$ yo foal my-app
$ cd my-app
$ npm run dev:app

Features

TypeScript

Use type-checking and the latest ECMAScript features to increase your productivity.

Scalable

Start with a small app. Then scale your code as your project grows up.

Testable

Testing your code shouldn't be difficult. Foal components are designed to easily be tested.

High-level components

Foal architecture brings you structure to develop complex apps. Start now playing with its controllers, services, modules and hooks.

Progressive migration

Already having an express app running on prod? No worries, you can progressively move your code from express to FoalTS.

Great tools

Use the foal generator to quickly start an app and then write and build code.
