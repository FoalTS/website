TypeScript

Increase your productivity with JavaScript that scales.

TypeScript brings you optional static type-checking along with the latest ECMAScript features. Writing FoalTS with TypeScript has been and will always be fundamental. Code is more elegant and concise. Most of the silly mistakes are caught at compilation. And autocompletion is well-handled and the API is better documented.

Essential Components

Do not waste time reinventing the wheel.

Some patterns always come back while developing a web app. Set up a REST API, talk to the database or manage authentication shouldn't be a hard task. FoalTS lets you focus on the business logic, not on re-inventing the wheel.

Development Tools

Focus on coding, not on setting up an environment.

Every project needs dev tools. But setting up and maintaining such an environment in Node.Js is often hard or time consuming. FoalTS focuses on helping the developer to code. Generators, watchers, the compiler, linter and test framework are all provided in FoalTS. No more set up, get started right away.

FoalTS enhances the developer experience by providing the tools and components essential to creating a Web application.
