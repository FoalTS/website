FoalTS

A Node.Js framework for ambitious web apps.

Get started

Create your first app

$ npm install -g yo generator-foal
$ yo foal my-app
$ cd my-app
$ npm run compile
$ npm run start

Features

TypeScript

Use optional static type-checking along with the latest ECMAScript features thanks to TypeScript.

Scalable

Foal architecture lets you start coding small apps which will be easily extended as your team and needs grow up.

Testable

Testing your code shouldn't be difficult. Foal components are designed to easily be tested.

High-level components

Complex apps need structure. Enjoy coding with foal controllers, services, modules and hooks.

Progressive integration

Already having an express app running on prod? No worries, you can progressively move your code from express to FoalTS.

Great tools

Use the foal generator to quickly start an app and then write and build its code.
