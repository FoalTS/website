Documentation

Basics

Services

Singleton classes which serve a restricted and well-defined purpose.

Controllers

Sub-category of services which aims to be attached to the request handler.

Modules

Chunks dividing the app.

Pre-hooks

Decorators to perform basic tasks of request handling.

Packages

@foal/authorization

Pre-hooks for access control.

@foal/core

Core package of the framework.

@foal/ejs

View services to render ejs templates.

@foal/express

Express adaptater to bootstrap the app.

@foal/sequelize

Database connections and CRUD services.
